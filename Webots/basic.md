## 视角操作

**常用操作**:

| 操作     | 功能           | 备注       |
| -------- | -------------- | ---------- |
| 鼠标左键 | 旋转视角       | 针对选定点 |
| 滚轮     | 缩放视角       |            |
| 按下滚轮 | 难用的旋转视角 |            |
| 鼠标右键 | 平移视角       |            |

**规范视角**：

使用控制界面右上角的视角选择按钮，进行视角选择；
